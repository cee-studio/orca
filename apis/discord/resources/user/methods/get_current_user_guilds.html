Get Current User Guilds#

ORCAcode discord_get_current_user_guilds(struct discord *client, struct discord_guild ***ret)#

Get guilds client is a member of.

Note

Requires the guilds oauth2 scope

Parameters:

  • client – the client created with discord_init()

  • ret – if successful, a null-terminated list of discord_guild, that should be free’d with discord_guild_list_free()

Returns:

ORCAcode for how the operation went, ORCA_OK means nothing out of the ordinary
